This application aims to prove that a Computer Vision Neural Network can be used to power a WebAR/Mobile-AR application. The finished app will guide the user through the preparation of a fruit salad given using a built in recipe (step-by-step format), where the neural network knows where to display the 3D-Augmentations by locating the ingredients & tools (object detection). This is a part of my MSc Thesis. It currently exists in two git repos, a complete rewrite of what has been developed thus far is in the works.
